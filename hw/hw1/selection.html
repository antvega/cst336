Movement of selection sort algorithm:

The algorithm starts at the beginning of the array. Each element is compared with the next. A "smallest" variable is saved and is compared to the rest. If there is a smaller element, that element becomes the smallest. At the end of the list, the smallest variable is swapped with the index the loop started with. The time complexity is n^2.
